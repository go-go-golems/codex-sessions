# Codex Session CLI

The `codex-session` CLI helps you list, search, and inspect Codex session archives
stored as JSONL files. It provides fast, index-backed listing and search, and
offers commands for exporting and reflection workflows.

Use `codex-session help` to browse topics and commands in the interactive help UI.
